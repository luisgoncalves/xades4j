AlgorithmsParametersMarshallingProvider

xades4j.xml.marshalling.algorithms.AlgorithmsParametersMarshallingProvider

Class Overview

Marshaller of parameters of any algorithm. It it responsible for finding and invoking the appropriate parameters marshaller for each algorithm.

This interface is intended for use on internal components and may be subject to changes.
